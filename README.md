# Программа расчета сменного задания

Разработано для ООО «Новокуйбышевский завод масел и присадок» (НЗМП) в рамках производственной практики.

**Функционал:**
- Ввод названия продукта, планового и фактического количества
- Расчет процента выполнения плана
- Определение результата (план выполнен/не выполнен)
- Сохранение результатов в файл

**Используемые технологии:** Python 3
